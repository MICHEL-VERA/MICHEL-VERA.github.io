Compétences

Ce que j'utilise dans mon travail
HTML5
C'est ce qui crée le cadre de votre site web ou de votre application, et la version cinq me permettra de créer une structure plus optimisée pour le référencement de votre produit.
CSS3
Ce langage de style me permet de créer absolument n'importe quelle apparence pour votre site web ou votre application. Tout n'est limité que par votre imagination!
Java Script
Ce langage de programmation vous permet d'animer n'importe quoi : curseurs, fenêtres, infobulles, onglets, réception de données du serveur, etc.
Jquery
La bibliothèque Jquery accélérera le développement. Nous ne l'intégrerons pas dans le projet sans nécessité, mais nous avons les compétences pour travailler avec.
React
Cette bibliothèque vous permet de créer des applications-web. Nous pouvons créer le produit le plus interactif possible en fonction de vos besoins.
Node.js
Cette plateforme vous permet de créer un backend pour votre produit - le "cerveau" qui effectuera les actions que l'utilisateur ne voit pas
Mongo.db
Il s'agit d'une base de données non relationnelle qui stocke les données de votre site web ou de votre application
